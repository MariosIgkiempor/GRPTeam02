About the Project
The Machine Learning Dataset Parsing tool takes a dataset in CSV format and works out the best machine leanring method to use for that dataset. Click on the datasets page to view all public datasets, or log in to upload your own private datasets. This application is suitable for teachers as a teaching aid, as well as anyone wanting to start learning machine learning methods.
